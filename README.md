# Library Management System

A simple CRUD application for managing users, books, and their respective loans.  
Developed as part of the Programming Language III course, this project simulates a basic library system that allows creating, updating, listing, and deleting records efficiently.

---

## Sistema de Gerenciamento de Biblioteca

Aplicação CRUD simples para gerenciar usuários, livros e seus respectivos empréstimos.  
Desenvolvido como parte da disciplina de Linguagem de Programação III, este projeto simula um sistema básico de biblioteca, permitindo criar, atualizar, listar e excluir registros de forma prática e organizada.
